BrightTV Analytics Dashboard README Project Overview BrightTV aims to grow its subscription
base and better understand viewer behaviour. This dashboard analyzes the platform's user profiles,
viewing sessions, and content trends to deliver actionable insights for the CVM team. Dashboard
Structure 1. Executive Overview
2. Audience & User Insights
3. Content & Consumption Analytics
4. Strategic Insights & Recommendations
Dataset Information File: Bright tv dataset.xlsx
Fields: UserID, Gender, Race, Age, Province, Channel2, RecordDate2 (UTC)
Calculated Fields SA_Time = RecordDate2 + 2 hours
Date, Hour, Weekday extracted
Sessions per User = Total Sessions / Unique Users
Filters Date range, Province, Gender, Age group, Channel, Category
Design Guidelines Dark theme, glassmorphism, neon accents, 12-column layout
